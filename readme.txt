One time steps:
1) Once you are in this folder, open cmd and type "npm install"

2) After completion, go to backend/venv and open cmd and type "pip install -r requirements.txt"

Steps while running code:
1) Go to venv folder and type "set FLASK_APP=server.py"

2) After that run "flask run"

3) Now go to main folder and type "ng serve"

Website url:
http://localhost:4200/home (this can also be found after executing step 3)
